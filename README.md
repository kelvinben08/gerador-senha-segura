# Gerador de Senha Segura
- Programa em Python que gera senhas seguras aleatórias para uso em sistemas de cadastro de usuários.

---

### Funcionalidades
- Gera senhas com 12 caracteres
- Inclui letras maiúsculas e minúsculas
- Inclui números
- Inclui caracteres especiais
- Garante que todos os tipos obrigatórios estejam presentes

---

### Como executar

```bash
python main.py
```

---

### Exemplo de saída
- Senha gerada: A7b@X2m#Q9p$

---

### Tecnologias

Python 3
